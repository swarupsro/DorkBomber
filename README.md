# DorkBomber

A single-page passive OSINT launcher for authorized bug bounty reconnaissance.

Open `index.html` in a browser, enter an in-scope target domain, choose a mission, and launch or copy target-scoped search pivots. The default Endpoint Mapper workflow focuses on common routes such as API docs, auth flows, admin panels, upload/download handlers, callbacks, source maps, and staging hosts.

DorkBomber does not scrape Google or scan hosts. Use it only for assets you are authorized to test.
